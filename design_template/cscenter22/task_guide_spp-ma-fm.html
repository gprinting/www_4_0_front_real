경인쇄

양식지
Form Printing
반드시 작업 시 주의사항, 작업규약을 반드시 확인하시고 주문해주세요.
주문한 데이터는 당사가 편집/수정 작업을 할 수 없으며, 작업규약에 맞지 않는 데이터로 인한 오류 건은 사고 처리가 되지 않습니다.
  1. 양식지는 권단위로 주문합니다. 1권당 100매 철로 떡제본됩니다.
  2. 정매생산을 원하실 경우 여분의 종이가 더 필요하며, 별도의 요금이 추가로 발생합니다.
  3. 아미가 20%이하일 경우 인쇄물이 깨끗하게 생산되지 않을 수 있습니다. 이로 인한 재인쇄는 불가합니다.
  4. 마감시간은 1차 오전 11시, 2차 오후5시입니다.
  5. 출고기간은 2~4일이며 후가공이 많을 경우 출고일자는 늦춰질 수 있습니다.
  6. 입금 완료 후 데이터 확인이 가능합니다. 등급별 할인율이 다르니 결제금액을 꼭 확인해주십시오.
  7. 고객의 데이터는 수정하지 않는 것이 원칙입니다. 조판중 이전 단계에서는 마이페이지에서 데이터 재전송, 주문취소가 가능합니다. (접수중, 접수검토중일 때는 취소가 불가합니다.)
  8. 주문서 작성 시 장바구니에서 묶음배송 및 선불택배를 신청할 수 있습니다.
  9. 묶음배송을 신청할 때는 후가공 여부를 확인해야 합니다. 묶음배송을 신청한 다른 제품도 배송이 늦어질 수 있습니다.

인쇄용 파일 만들기
  • 디자인 작업을 할 때 작업사이즈는 재단사이즈보다 사방으로 5mm씩 작게 해야 됩니다.
  • 규격사이즈만 제작이 가능합니다.
  • 재단하는 과정에서 밀림현상이 생기므로 재단사이즈보다 5mm 이상 안쪽으로 디자인해야 안전합니다.
  • 1도의 경우에는 K100으로 작업하고, 아미 20%이하는 인쇄되지 않을 수 있습니다.
  • 2도의 경우 색상별로 분판하고 색상을 적습니다.
  • 해당 절수의 템플릿을 다운받고 색상, 제본, 앞뒤여부를 체크하여 데이터와 같이 업로드합니다.
  • 제본방향을 지정할 때는 인쇄물을 정면으로 보고 상철과 좌철을 지정하십시오.
  • 양면의 경우 앞뒤 배열이 머리/머리인지, 머리/꼬리인지 확인합니다.
  • 문서의 색상은 CMYK로 되어 있고, 글꼴은 곡선화(아웃라인)를 해서 보내주셔야 합니다.
  • 재단선 및 편집가이드선은 삭제하지 마시고 꼭 안 보이는 선으로 작업합니다.
  • 그룹되지 않아서 빠지는 개체가 없도록 합니다.
  • 일러스트레이터나 인디자인을 이용한 디자인일 경우 이미지 링크여부를 확인합니다.
  • 링크되어 있는 이미지는 파일에서 반드시 이미지포함(EMBEDED)하여 저장합니다.
  1. 일러스트레이터
    • [문자] → [윤곽선 만들기]를 사용하여 글꼴을 아웃라인(shift+ctrl+O)으로 만듭니다.
    • [윈도우] → [연결] 창을 띄워 링크된 이미지가 있으면 포함(embeded) 시켜줍니다.
    • 필요없는 레이어는 잠금이나 숨김 없이 삭제해주셔야 합니다.
    • (출력시 필요없는 레이어가 출력될 수 있습니다.)
  2. 코렐드로우
    • [배열] → [곡선으로 변환](ctrl+Q)을 사용하여 글꼴을 곡선처리 합니다.
    • [파일] → [문서 등록 정보] 창을 띄워 글꼴, 색상, 효과 등을 살펴봅니다.
  3. 포토샵
    • 여러 레이어일 경우 배경으로 이미지 병합을 해야 글꼴도 이미지화 되고 레이어도 하나로 합쳐져서 저장할 수 있습니다.
    • 300dpi, 재단선 보이지 않게 저장합니다.
  4. 애크러뱃
    • [파일매뉴] → [속성](ctrl+D)을 사용하여 글꼴이 사용 되었는지 확인합니다.
    • 사용된 글꼴이 있다면 원본파일에서 곡선화하여 재저장 후 속성에서 다시 글꼴 사용여부를 확인합니다.

단면이 여러 건일 경우
  • 3개의 파일을 만들어 한 폴더에 넣은 후 압축합니다.
세로형의 경우
  • 세로형은 가로형으로 회전하여 파일을 만듭니다.
  • 앞면은 90도 회전, 뒷면은 -90도 회전해야 압뒤가 맞게 됩니다.
  • 만들어진 파일을 한 폴더에 넣은 후 압축합니다.
양면이 여러 건일 경우
  • 포토샵은 앞면1, 뒷면1, 앞면2, 뒷면2, 앞면3, 뒷면3,
    6개의 파일을 만들어 한 폴더에 넣은 후 압축합니다.
  • PDF의 경우 앞면, 뒷면을 하나의 파일로 만듭니다.
    3개의 파일을 만들어 한 폴더에 넣은 후 압축합니다.
작업 시 유의사항
  1. 후가공
      후가공 제약사항
      떡제본 100매철
      미싱 80g이상
      넘버링 6자리(26×8mm)적색
  2. 2도 인쇄
    • 한 파일에 검정색과 청색이 인쇄될 부분을 분판해줍니다.
    • 4×6 계열만 배다인쇄가 가능합니다.
  3. 양면인쇄
    • 낱장으로 뜯었을 때 좌우로 넘기면 인쇄물 위치가 달라집니다. 내용에 맞게 체크해주세요.
  4. 주문기준
    • 종이는 70g,80g100g 모조만 가능합니다.
    • 2도인쇄까지 지정된 5가지 색상만 인쇄됩니다.
    • 제본은 100매철 기준이며 낱장도 가능합니다.
