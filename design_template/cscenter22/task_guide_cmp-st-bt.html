스티커

기본 도무송 스티커
Basic Thomson Sticker
반드시 작업 시 주의사항, 작업규약을 반드시 확인하시고 주문해주세요.
주문한 데이터는 당사가 편집/수정 작업을 할 수 없으며, 작업규약에 맞지 않는 데이터로 인한 오류 건은 사고 처리가 되지 않습니다.
  1. 입금 완료 후 데이터 확인이 가능합니다. 등급별 할인율이 다르니 결제금액을 꼭 확인해주십시오.
  2. 고객의 데이터는 수정하지 않는 것이 원칙입니다. 조판중 이전 단계에서는 마이페이지에서 데이터 재전송, 주문취소가 가능합니다. (접수중, 접수검토중일 때는 취소가 불가합니다.)
  3. 당일판은 강접코팅스티커만 가능하며 2,000장 이상은 신청할 수 없습니다.
  4. 스티커 접착제는 휘발성이므로 장시간 노출시 접착성은 감소되며, 표면에 굴곡이 있을 경우 잘 떨어집니다.
  5. 스티커는 기본코팅, 도무송 이외에 별도로 다른 후가공이 진행되지 않습니다.
  6. 당일판 품목별 마감시전 이전에 입금된 주문건에 한해서만 제작이 진행됩니다.
  7. 품목별 템플릿을 다운받아 작업양식에 맞게 디자인바랍니다. 기존과 다르게 칼선이 변경된 것이 있으니 새롭게 다운받으시길 권장합니다.
재단 및 작업사이즈
  • 작업선과 재단선은 투명하게 작업하여 인쇄에서 보이지 않도록 하며, 그룹을 지어주시기 바랍니다.
  • 별 사이즈로 작업하실 경우에도 사방 1mm씩 여분을 주어 작업해주십시오
재단 및 작업사이즈
다양한 규격사이즈 본 제품은 배다(배경)의 선택에 따라 작업사이즈가 달라집니다.
사이즈 확인은 주문 상세페이지를 확인해주세요.
비규격 본 제품은 비규격사이즈 제작이 가능합니다.
인쇄용 파일 만들기
  • 칼선과 칼선사이는 5mm간격으로 배열하고, 배다는 3mm씩 크게 합니다. 
  • 칼선을 지우지 마시고, 안 보이는 선으로 만든 후 칼선과 데이터를 그룹지어 보냅니다.
  • 칼선파일은 일러스트레이터나 코렐드로우만 가능합니다. 포토샵 이미지라도 칼선은 벡터파일(일러,코렐)만 가능합니다.
  • 최소 칼 크기는 10mm이며 너무 복잡한 모양이나 작은 크기의 디자인은 제작이 불가능할 수 있습니다.
  • 포토샵이나 PDF로 작업할 때는 1개의 파일이 칼선에 해당하는 재단사이즈로 맞추고 칼선을 보이지 않게 합니다.
  • 문서의 색상은 CMYK로 되어 있고, 글꼴은 곡선화(아웃라인)를 해서 보내주셔야 합니다.
  • 복잡한 개체가 특수한 효과를 사용한 것은 비트맵처리(레스터화)를 해야 출력이 안전하게 나옵니다.
  • 재단선 및 편집가이드선은 삭제하지 마시고 꼭 안 보이는 선으로 작업합니다.
  • 그룹되지 않아서 빠지는 개체가 없도록 합니다.
  • 일러스트나 인디자인일 경우 사용한 이미지가 있으면 링크여부를 확인합니다.
  • 링크되어 있는 이미지는 파일에서 이미지포함(EMBEDED)하여 저장합니다.
  1. 일러스트레이터
    • [문자] → [윤곽선 만들기]를 사용하여 글꼴을 아웃라인(shift+ctrl+O)으로 만듭니다.
    • [윈도우] → [연결] 창을 띄워 링크된 이미지가 있으면 포함(embeded) 시켜줍니다.
    • 필요없는 레이어는 잠금이나 숨김 없이 삭제해주셔야 합니다.
    • (출력시 필요없는 레이어가 출력될 수 있습니다.)
  2. 코렐드로우
    • [배열] → [곡선으로 변환](ctrl+Q)을 사용하여 글꼴을 곡선처리 합니다.
    • [파일] → [문서 등록 정보] 창을 띄워 글꼴, 색상, 효과 등을 살펴봅니다.
  3. 포토샵
    • 여러 레이어일 경우 배경으로 이미지 병합을 해야 글꼴도 이미지화 되고 레이어도 하나로 합쳐져서 저장할 수 있습니다.
    • 300dpi, 재단선 보이지 않게 저장합니다.
  4. 애크러뱃
    • [파일매뉴] → [속성](ctrl+D)을 사용하여 글꼴이 사용 되었는지 확인합니다.
    • 사용된 글꼴이 있다면 원본파일에서 곡선화하여 재저장 후 속성에서 다시 글꼴 사용여부를 확인합니다.

  • 3개의 파일을 만들어 한 폴더에 넣은 후 압축합니다.
세로형의 경우
  • 세로형은 가로형으로 회전하여 파일을 만듭니다.
  • 앞면은 90도 회전, 뒷면은 -90도 회전해야 압뒤가 맞게 됩니다.
  • 만들어진 파일을 한 폴더에 넣은 후 압축합니다.
양면이 여러 건일 경우
  • 포토샵은 앞면1, 뒷면1, 앞면2, 뒷면2, 앞면3, 뒷면3,
    6개의 파일을 만들어 한 폴더에 넣은 후 압축합니다.
  • PDF의 경우 앞면, 뒷면을 하나의 파일로 만듭니다.
    3개의 파일을 만들어 한 폴더에 넣은 후 압축합니다.
작업 시 유의사항
  1. 재단종류
    작업선 제거재단
    • 작업선이 보이지 않는 재단방법입니다.
    작업선 포함재단
    • 작업선이 보이는 재단방법입니다.
    • 작업선이란 제작과정에서 재단을 하기 위해 임의로 넣은 선을 말합니다.
  2. 재단 방식
    도무송이 1개인 경우
    • 도무송 선의 가장자리 사이즈가 도무송 사이즈가 됩니다.
    • 도무송(모양따기)는 작업물 모양대로 재단하는 방법을 말합니다.
    도무송이 2개 이상인 경우
    • 도무송이 2개일 떄는 전체 도무송 선의 가장자리 사이즈가 도무송 사이즈가 됩니다.
    • 또한, 도무송과 도무송 사이의 간격이 6mm(각각3mm)여야 합니다.
    도무송이 여러개인 경우
    • 다수의 도무송이 있는 경우 전체 도무송을 합친 가장자리 사이즈가 도무송 사이즈가 됩니다.
    • 또한, 도무송과 도무송 사이의 간격이 6mm(각각3mm)여야 합니다.
  3. 파일 저장 방법
    일러스트레이터/코렐드로우
    • 일러스트레이터 및 코렐로 작업하신 경우 4도 칼라인쇄 파일과 도무송 파일을 배열합니다. 2건 이상 접수시에는 데이터를 세로로 추가 배치하여 접수해주세요.
    PDF 파일 저장 방법
    • 작업사이즈에 맞는 문서를 생성합니다. 4도칼라 인쇄파일을 1페이지, 도무송 파일을 2페이지에 배열합니다. 2건 이상 접수시에는 3페이지부터 같은 방법으로 아래로 배열하면 됩니다.
    포토샵 파일 저장 방법
    • 포토샵의 파일 해상도를 300dpi로 설정한 후 템플릿 작업사이즈의 크기에 맞춰 파일을 생성합니다.
    • 4도 칼라인쇄 파일과 도무송 파일을 저장합니다. 도무송 파일은 일러스트레이터 및 코렐 형식으로 같이 주셔야 합니다.
    • 모든 파일은 한 폴터에 모아 Zip으로 압축해주시기 바랍니다.
