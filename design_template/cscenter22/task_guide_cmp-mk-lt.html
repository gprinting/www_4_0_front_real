반드시 작업 시 주의사항, 작업규약을 반드시 확인하시고 주문해주세요.
주문한 데이터는 당사가 편집/수정 작업을 할 수 없으며, 작업규약에 맞지 않는 데이터로 인한 오류 건은 사고 처리가 되지 않습니다.
  1. 인쇄물 색상이 너무 진하면 재단 후 뒷묻음이 생길 수 있습니다. 이로 인한 재인쇄는 불가합니다.
  2. 실크가공 등의 공정이 추가되는 제품으로 기본 출고일이 3박4일입니다. 당일판 주문이 불가합니다.
  3. 입금 완료 후 데이터 확인이 가능합니다. 등급별 할인율이 다르니 결제금액을 꼭 확인해주십시오.
  4. 고객의 데이터는 수정하지 않는 것이 원칙입니다. 조판중 이전 단계에서는 마이페이지에서 데이터 재전송, 주문취소가 가능합니다. (접수중, 접수검토중일 때는 취소가 불가합니다.)
  5. 품목별 템플릿을 다운받아 작업양식에 맞게 디자인바랍니다. 기존과 다르게 칼선이 변경된 것이 있으니 새롭게 다운받으시길 권장합니다.
  6. 주문서 작성 시 장바구니에서 묶음배송 및 선불택배를 신청할 수 있습니다.
  7. 묶음배송을 신청할 때는 후가공 여부를 확인해야 합니다. 묶음배송을 신청한 다른 제품도 배송이 늦어질 수 있습니다.
재단 및 작업사이즈
  • 작업선과 재단선은 투명하게 작업하여 인쇄에서 보이지 않도록 하며, 그룹을 지어주시기 바랍니다.
  • 별 사이즈로 작업하실 경우에도 사방 1mm씩 여분을 주어 작업해주십시오
재단사이즈 작업사이즈
복권 86 × 52 mm 88 × 54 mm
인쇄용 파일 만들기
  • 디자인 작업을 할 때 작업사이즈는 재단사이즈보다 사방으로 1mm씩 크게 해야 됩니다.
  • 실크가공 할 때 약간의 밀림이 방생하기 때문에 실크판을 만들때 가려질 내용보다 2~3mm 이상 크게 만들어야 합니다. 또한 복잡한 모양은 불가하며, 가리지 말아야 할 내용과는 거리를 두고 디자인 해야 안전합니다.
  • 인쇄할 데이터와 후가공에 사용되는 실크판을 만듭니다. 이 때 인쇄물에는 보이지 않도록 작업하며 같은 위치에 K100으로 된 박스를 첨부합니다.
  • 실크는 한면만 가능하고, 실크면을 앞면으로 배치합니다.
  • 후가공 파일은 벡터파일 (일러스트나 코렐드로우)로 만들어야 합니다. 포토샵이미지나 기타 프로그램으로 작업된 파일은 사용할 수 없습니다. 
  • 재단하는 과정에서 밀림현상이 생기므로 재단사이즈보다 3mm 이상 안쪽으로 디자인해야 안전합니다.
  • 문서의 색상은 CMYK로 되어 있고, 글꼴은 곡선화(아웃라인)를 해서 보내주셔야 합니다.
  • 복잡한 개체가 특수한 효과를 사용한 것은 비트맵처리(레스터화)를 해야 출력이 안전하게 나옵니다.
  • 재단선 및 편집가이드선은 삭제하지 마시고 꼭 안 보이는 선으로 작업합니다.
  • 앞면, 뒷면 각 데이터끼리 그룹을 합니다. 그룹되지 않아서 빠지는 개체로 인한 재인쇄는 불가합니다.
  1. 일러스트레이터
    • [문자] → [윤곽선 만들기]를 사용하여 글꼴을 아웃라인(shift+ctrl+O)으로 만듭니다.
    • [윈도우] → [연결] 창을 띄워 링크된 이미지가 있으면 포함(embeded) 시켜줍니다.
    • 필요없는 레이어는 잠금이나 숨김 없이 삭제해주셔야 합니다.
    • (출력시 필요없는 레이어가 출력될 수 있습니다.)
  2. 코렐드로우
    • [배열] → [곡선으로 변환](ctrl+Q)을 사용하여 글꼴을 곡선처리 합니다.
    • [파일] → [문서 등록 정보] 창을 띄워 글꼴, 색상, 효과 등을 살펴봅니다.
  3. 포토샵
    • 여러 레이어일 경우 배경으로 이미지 병합을 해야 글꼴도 이미지화 되고 레이어도 하나로 합쳐져서 저장할 수 있습니다.
    • 300dpi, 재단선 보이지 않게 저장합니다.
  4. 애크러뱃
    • [파일매뉴] → [속성](ctrl+D)을 사용하여 글꼴이 사용 되었는지 확인합니다.
    • 사용된 글꼴이 있다면 원본파일에서 곡선화하여 재저장 후 속성에서 다시 글꼴 사용여부를 확인합니다.
데이타배열일러스트 / 코렐드로우
한 대지에 넘치지 않게 데이터를 배열합니다.
단면이 여러 건일 경우
  • 앞면 기준으로 아래로 배열합니다.
양면이 여러 건일 경우
  • 앞면 왼쪽, 뒷면 오른쪽 기준으로 아래로 배열합니다.
세로형이 여러 건일 경우
  • 세로형은 가로형으로 회전하여 배열합니다.
  • 앞면은 90도, 뒷면은 -90도 회전해야 앞뒤가 맞게 됩니다.
  • 앞면 왼쪽, 뒷면 오른쪽 기준으로 아래로 배열합니다.
데이타배열포토샵 / PDF(인디자인)
한 대지에 한 개의 데이터를 만들어서 압축하여 보냅니다.
단면이 여러 건일 경우
  • 3개의 파일을 만들어 한 폴더에 넣은 후 압축합니다.
세로형의 경우
  • 세로형은 가로형으로 회전하여 파일을 만듭니다.
  • 앞면은 90도 회전, 뒷면은 -90도 회전해야 압뒤가 맞게 됩니다.
  • 만들어진 파일을 한 폴더에 넣은 후 압축합니다.
양면이 여러 건일 경우
  • 포토샵은 앞면1, 뒷면1, 앞면2, 뒷면2, 앞면3, 뒷면3,
    6개의 파일을 만들어 한 폴더에 넣은 후 압축합니다.
  • PDF의 경우 앞면, 뒷면을 하나의 파일로 만듭니다.
    3개의 파일을 만들어 한 폴더에 넣은 후 압축합니다.
작업 시 유의사항
  1. 복권 작업 시 배치 및 그룹
    • 그룹은 반드시 지어주셔야 하며, 앞면과 뒷면 각각 그룹을 지어줍니다.
    • 가로형 복권은 좌우로 배열, 세로형 복권은 머리와 머리를 붙여 좌우로 배열합니다.
  2. 복권 주문 수량
    • 500장 단위로만 주문할 수 있습니다.(1장 필요시에도 500장을 주문하셔야 합니다.)
    • 전체수량 선택 후 실크인쇄될 부분(등수 및 내용)을 입력합니다.
