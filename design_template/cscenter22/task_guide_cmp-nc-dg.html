Digital Card
반드시 작업 시 주의사항, 작업규약을 반드시 확인하시고 주문해주세요.
주문한 데이터는 당사가 편집/수정 작업을 할 수 없으며, 작업규약에 맞지 않는 데이터로 인한 오류 건은 사고 처리가 되지 않습니다.
  1. 인쇄물 색상이 너무 진하면 재단 후 뒷묻음이 생길 수 있습니다. 이로 인한 재인쇄는 불가합니다.
  2. 입금 완료 후 데이터 확인이 가능합니다. 등급별 할인율이 다르니 결재금액을 꼭 확인해주십시오.
  3. 고객의 데이터는 수정하지 않는 것이 원칙입니다. 조판중 이전 단계에서는 (접수중, 접수검토중일 때는 취소가 불가합니다.)
  4. 당일판은 후가공을 신청할 수 없으며, 당일판 품목별 마감시간 이전 입금된 주문건에 한해서만 진행됩니다.
  5. 품목별 템플릿을 다운받아 작업양식에 맞게 디자인바랍니다. 기존과 다르게 칼선이 변경된 것이 있으니 새롭게 다운받으시길 권장합니다.
  6. 주문서 작성 시 장바구니에서 묶음배송 및 선불택배를 신청할 수 있습니다.
  7. 묶음배송을 신청할 때는 후가공 여부를 확인해야 합니다. 묶음배송을 신청한 다른 제품도 배송이 늦어질 수 있습니다.
재단 및 작업사이즈
  • 작업선과 재단선은 투명하게 작업하여 인쇄에서 보이지 않도록 하며, 그룹을 지어주시기 바랍니다.
  • 별 사이즈로 작업하실 경우에도 사방 1mm씩 여분을 주어 작업해주십시오
재단사이즈 작업사이즈
규격 90 × 50 mm 92 × 52 mm
비규격 본 제품은 비규격사이즈 제작이 가능합니다.
인쇄용 파일 만들기
  • 디자인 작업을 할 때 작업사이즈는 재단사이즈보다 사방으로 1mm씩 크게 해야 됩니다.
  • 비규격사이즈도 재단이 가능하며, 40mm 이하는 재단이 되지 않습니다.
  • 고품격 명함의 경우 규격으로만 제작이 가능합니다.
  • 재단하는 과정에서 밀림현상이 생기므로 재단사이즈보다 3mm 이상 안쪽으로 디자인해야 안전합니다.
  • 문서의 색상은 CMYK로 되어 있고, 글꼴은 곡선화(아웃라인)를 해서 보내주셔야 합니다.
  • 복잡한 개체가 특수한 효과를 사용한 것은 비트맵처리(레스터화)를 해야 출력이 안전하게 나옵니다.
  • 재단선 및 편집가이드선은 삭제하지 마시고 꼭 안 보이는 선으로 작업합니다.
  • 앞면, 뒷면 각 데이터끼리 그룹을 합니다. 그룹되지 않아서 빠지는 개체로 인한 재인쇄는 불가합니다.
  • 데이터가 양면 또는 2개 이상일 경우 아래 예시대로 하셔야 앞뒷면이 바뀌지 않습니다.
  • 일러스트레이터나 인디자인을 이용한 디자인일 경우 이미지 링크여부를 확인합니다.
  • 링크되어 있는 이미지는 파일에서 반드시 이미지포함(EMBEDED)하여 저장합니다.
  • 후가공 파일은 벡터파일 (일러스트나 코렐드로우)로 만들어야 합니다. 포토샵이미지나 기타 프로그램으로 작업된 파일은 사용할 수 없습니다.
  1. 일러스트레이터
    • [문자] → [윤곽선 만들기]를 사용하여 글꼴을 아웃라인(shift+ctrl+O)으로 만듭니다.
    • [윈도우] → [연결] 창을 띄워 링크된 이미지가 있으면 포함(embeded) 시켜줍니다.
    • 필요없는 레이어는 잠금이나 숨김 없이 삭제해주셔야 합니다.
    • (출력시 필요없는 레이어가 출력될 수 있습니다.)
  2. 코렐드로우
    • [배열] → [곡선으로 변환](ctrl+Q)을 사용하여 글꼴을 곡선처리 합니다.
    • [파일] → [문서 등록 정보] 창을 띄워 글꼴, 색상, 효과 등을 살펴봅니다.
  3. 포토샵
    • 여러 레이어일 경우 배경으로 이미지 병합을 해야 글꼴도 이미지화 되고 레이어도 하나로 합쳐져서 저장할 수 있습니다.
    • 300dpi, 재단선 보이지 않게 저장합니다.
  4. 애크러뱃
    • [파일매뉴] → [속성](ctrl+D)을 사용하여 글꼴이 사용 되었는지 확인합니다.
    • 사용된 글꼴이 있다면 원본파일에서 곡선화하여 재저장 후 속성에서 다시 글꼴 사용여부를 확인합니다.

단면이 여러 건일 경우
  • 3개의 파일을 만들어 한 폴더에 넣은 후 압축합니다.
세로형의 경우
  • 세로형은 가로형으로 회전하여 파일을 만듭니다.
  • 앞면은 90도 회전, 뒷면은 -90도 회전해야 압뒤가 맞게 됩니다.
  • 만들어진 파일을 한 폴더에 넣은 후 압축합니다.
양면이 여러 건일 경우
  • 포토샵은 앞면1, 뒷면1, 앞면2, 뒷면2, 앞면3, 뒷면3,
    6개의 파일을 만들어 한 폴더에 넣은 후 압축합니다.
  • PDF의 경우 앞면, 뒷면을 하나의 파일로 만듭니다.
    3개의 파일을 만들어 한 폴더에 넣은 후 압축합니다.
작업 시 유의사항
  1. 명함 공통 유의사항
    • 동일한 데이터일 경우에도 색상이 진하거나 연하게 제작될 수 있습니다.
    • 배다가 있는 데이터는 작업사이즈에 맞춰 정확히 배다를 맞춰야 재단이 깔끔하게 됩니다.
    • 재단시 1~2mm의 오차는 발생할 수 있습니다. 따라서  안쪽으로 3~5mm의 정도의 여분을 두는 것이 안전합니다.
    • 인쇄물의 경우 색상은 CMYK를 기본으로 작업해주십시오.
    • RGB, Pantone, DIC, 사용자 정의 색상 등을 사용하시는 경우 먹(K100%)으로 인쇄되거나 의도하지 않은 색상으로 결과물이 제작될 수 있습니다. 이로 인한 재작업은 불가합니다.
    • 배다가 진하게 작업된 명함은 뒷묻음이 발생할 수 있으니 작업 시 고려하시기 바랍니다.
    • 색상사용에 있어 화면의 색상으로만 판단하는 것은 위험합니다. 화면의 색상과 결과물의 색상에서 차이가 발생하는 일이 많습니다. 이로 인한 재작업은 불가합니다.
    • 그룹이 되지 않아 발생한 인쇄사고로 인한 재작업은 불가합니다.
    • 사방에 간격이 동일한 선이 들어간 작업물의 경우 재단에서 밀림현상에 의해 디자인 의도와 다르게 제작될 수 있습니다. 이를 충분히 인지하시고 작업하시기 바랍니다.
  2. 곡선처리
    • 문자는 반드시 도안문자를 사용하시고(단락문자열은 곡선처리 되지 않음) 반드시 서체는 곡선처리(ctrl+Q) 하십시오.
    • 코렐드로우 13버전부터 단락문자열도 곡선처리가 가능합니다.
    • 재단선 표시(톰보)는 넣지 마시기 바랍니다.
  3. 테두리 색/두께
    • 작업테두리에 두께를 주면 재단시 밀림 현상이 있습니다. 정확히 재단할 경우에도 선이 보이므로  테두리선은 반드시 0point로 작업합니다.
    • IBM 주문시 '색깔채움없음'으로 작업해주십시오.
    • 작업선과 재단선 외곽선 색상을 제거해주십시오.
    • 재단선에 너무 가깝게 작업하실 경우 내용부분이 재단되어 삭제될 수 있습니다. 반드시 재단선에서 3~5mm여유를 두고 작업해주십시오.
  4. 배치방법
    1. 양면의 경우 앞면, 뒷면을 각각 따로 그룹 지어주세요.
    2. 여러 건이 있을 경우도 각각 분리해서 그룹 지어주세요.
    3. 앞면, 뒷면을 각각 그룹 지어준 뒤 데이터는 왼쪽에는 앞면, 오른쪽에는 뒷면을 배열해주세요.
      세로형 명함의 경우 앞면은 90도, 뒷면은 -90도 회전후 빈박스로 앞면, 뒷면을 0도로 맞춘 후 접수해주세요.
    • 단면인 경우, 여러 데이터를 위에서 아래로 배열해야 합니다. 좌우로 배열할 경우 양면으로 잘못 인쇄될 수 있습니다.
    • 양면인 경우, 앞면은 왼쪽, 뒷면은 오른쪽으로 배열합니다. 양면을 상하로 배열할 경우 단면으로 잘못 인쇄될 수 있습니다.

  5. 후가공 작업 시 유의사항
    박/ 형압 / 타공 / 복권
    • 금/은박, 형압, 타공, 복권을 만드실 때는 완성본과 후가공 부분이 빠진 4도, 후가공 지정 부분을 먹(K100)으로 지정한 것으로 세가지 데이터를 각각 나누고 그룹화하여 저장해 주십시오.
    • 박의 경우 박종류를 명기해 주시기 바랍니다.
    • 작업 상황에 따라 약간의 오차가 발생할 수 있음을 감안해 주시기 바랍니다.
    오시
    • 티켓이나 초대장을 만드는 경우, 오시나 미싱 등이 들어가게 되는데, 표시를 정확히 해 주십시오.
    • 별도의 돔보는 필요하지 않습니다. 재단 사이즈에서 1mm씩만 빼주시면 됩니다.
    • 실미싱이나 오시가 들어가는 티켓은 미싱이나 오시선이 앞면과 뒷면에 놓이는 위치가 반대여야 합니다. 그래야 인쇄 시 정확하게 잘리는 면이 같아집니다.
    미싱
    • 표지면은 180도 회전시켜서 위의 그림과 같이 위쪽에 붙여 주십시오.
    • 앞면, 뒷면, 각각 그룹을 지어서 앞면은 왼쪽, 뒷면은 오른쪽에 배열하십시오.
